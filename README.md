# Gestion de productos - WPF

#Descripcion:
Sistema de gestion de productos (ABML y busqueda) desarrollado con WPF y T-SQL(procedimientos almacenados).

#Arquitectura:
Arquitectura en 4 capas: Acceso a datos, Dominio con modelos, Servicios de entidades, y Vistas. 

#Diseño:
Diseño y modelado con T-SQL | Medologogia DataBaseFirst | Ingenieria inversa - Scaffold con Entity Framework

#Herramientas:
Visual Studio Community | SQL Server Management Studio
